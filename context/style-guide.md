## Color Palette

The application uses a defined color scheme:

- **Argentinian Blue**: `#5aa9e6` (primary brand color)
- **Cinereous**: `#987d7c` (secondary)
- **Black**: `#020202` (text/contrast)
- **Fern Green**: `#50723c` (accent)
- **Light Green**: `#a1e887` (success/highlight)
